# Software Requirements Specification (SRS) - Improved Requirements

## Corrected Requirements

1. **User-Friendly Interface**  
   The system shall provide a user interface that allows users to complete all main tasks within 3 clicks and without errors.

2. **Secure Student Records**  
   The system shall encrypt all student records using AES-256 encryption and backup data daily to ensure safety and integrity.

3. **Performance / Loading Speed**  
   The system shall load the dashboard within 2 seconds on standard hardware (Intel i5, 8GB RAM).

4. **User Account Management**  
   The system shall allow:  
   - User registration with email verification.  
   - Login using email/password.  
   - Profile management including updating personal info and changing passwords.

5. **Notifications**  
   The system shall send email notifications to users for registration confirmation, password reset, and upcoming deadlines within 5 minutes of triggering events.

# Test Plan

## Introduction
This document outlines the test strategy, test scope, and types of testing for the system.

## Scope
- Functional testing of user registration, login, and profile management.
- Performance testing of system loading speed.
- Security testing for student record storage.
- Notification delivery testing.

## Test Types
1. Unit Testing  
2. Integration Testing  
3. System Testing  
4. User Acceptance Testing

## Test Environment
- OS: Windows 10  
- Hardware: Intel i5, 8GB RAM  
- Browser: Chrome/Edge
